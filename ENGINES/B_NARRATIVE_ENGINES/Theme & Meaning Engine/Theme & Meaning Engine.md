# Theme & Meaning Engine
## Движок Тем и Смыслов

---

## 1) Identity / Идентификация

- Engine ID: TME-B6-006
- Class: Narrative Engine
- Level: B6 (Theme & Meaning Core)
- Status: Active
- Owner: Universe Engine

---

## 2) Purpose / Назначение

### EN
Defines thematic layers and meaning structures of a narrative.
Controls what a story is *about* on a conceptual level without dictating plot or characters.

### RU
Определяет тематические слои и структуры смысла повествования.
Управляет тем, *о чём* история на концептуальном уровне, не диктуя сюжет или персонажей.

---

## 3) Responsibilities / Ответственность

Responsible for:
- Theme definition frameworks
- Meaning layers and abstractions
- Thematic consistency rules
- Theme interaction and hierarchy
- Meaning validation across narrative

Отвечает за:
- Каркасы определения тем
- Слои смысла и абстракции
- Правила тематической согласованности
- Взаимодействие и иерархию тем
- Проверку целостности смыслов в повествовании

## 7) Operating Logic / Логика работы

1. Define primary and secondary themes
2. Establish meaning layers
3. Set thematic constraints and contrasts
4. Validate alignment with structure and arc
5. Ensure thematic consistency
6. Lock meaning framework

---

## 8) Constraints / Ограничения

This engine must NOT:
- Impose moral judgments
- Define plot events
- Control character psychology
- Override narrative structure
- Inject authorial intent directly
