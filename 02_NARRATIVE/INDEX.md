# Narrative
## Повествование

This module controls storytelling logic, plot construction,
dramatic structure and narrative flow.

Этот модуль управляет логикой повествования, построением сюжетов,
драматургией и потоком истории.

---

### Responsibility / Ответственность
- Plot structures
- Story arcs
- Conflict systems
- Narrative logic

Структуры сюжетов  
Арки историй  
Системы конфликтов  
Логика повествования
