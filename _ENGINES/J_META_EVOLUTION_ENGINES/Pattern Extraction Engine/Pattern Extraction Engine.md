## 2) Purpose / Назначение

### EN
Defines how recurring structures, behaviors, and relationships are detected across the system.
Extracts patterns without interpreting meaning or altering canon.

### RU
Определяет, как повторяющиеся структуры, поведения и связи выявляются внутри системы.
Извлекает паттерны без интерпретации смысла и без изменения канона.

---

## 3) Responsibilities / Ответственность

Responsible for:
- Pattern detection across layers
- Frequency and recurrence analysis
- Cross-domain similarity recognition
- Noise filtering
- Pattern candidate validation

Отвечает за:
- Выявление паттернов между слоями
- Анализ частоты и повторяемости
- Распознавание междоменного сходства
- Фильтрацию шума
- Валидацию кандидатов паттернов

1. Collect multi-layer data streams
2. Identify recurring structures and behaviors
3. Measure frequency and similarity
4. Filter noise and outliers
5. Score pattern relevance
6. Lock extracted pattern set
