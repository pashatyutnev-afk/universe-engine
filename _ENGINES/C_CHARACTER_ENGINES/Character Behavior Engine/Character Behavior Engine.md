# Character Behavior Engine
## Движок Поведения Персонажа

---

## 1) Identity / Идентификация

- Engine ID: CBE-C3-003
- Class: Character Engine
- Level: C3 (Character Behavior Core)
- Status: Active
- Owner: Universe Engine

---

## 2) Purpose / Назначение

### EN
Defines how a character translates internal states into observable actions.
Controls behavior patterns, reactions, and choices without creating narrative events.

### RU
Определяет, как внутренние состояния персонажа преобразуются в наблюдаемое поведение.
Управляет паттернами поведения, реакциями и выборами без создания сюжетных событий.

---

## 3) Responsibilities / Ответственность

Responsible for:
- Behavior models
- Reaction patterns
- Decision thresholds
- Action eligibility rules
- Consistency of behavioral output

Отвечает за:
- Модели поведения
- Паттерны реакций
- Пороги принятия решений
- Правила допустимости действий
- Согласованность поведенческого вывода

## 5) Inputs / Входные данные

Receives:

- World constraints
- Temporal context

---

## 6) Outputs / Выходные данные

Produces:

- Behavioral consistency indicators

---

## 7) Operating Logic / Логика работы

1. Read current psychological state
2. Evaluate motivation signals
3. Generate action candidates
4. Apply constraints and thresholds
5. Select behavioral response
6. Lock behavior consistency

---

## 8) Constraints / Ограничения

This engine must NOT:
- Create narrative events
- Write dialogue
- Assign moral values
- Override psychology or world laws
- Force plot progression
