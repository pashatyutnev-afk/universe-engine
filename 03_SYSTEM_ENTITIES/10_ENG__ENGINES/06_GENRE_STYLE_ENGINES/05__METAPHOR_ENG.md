---

## PURPOSE

Метафора — это “канал интерпретации”, который:
- объединяет сцены/темы
- помогает зрителю понять смысл
- не подменяет причинность (не магия объяснения)

---

## INPUTS

- Theme intent
- Symbol map
- Character lens
- World texture

---

## OUTPUTS

- Core metaphor statement (one)
- Supporting metaphors (0–2)
- Allowed imagery set
- Forbidden imagery set (to avoid mixed metaphors)

---

## METAPHOR PACKAGE (CANON)

- CORE METAPHOR: “X is like Y”
- WHY IT FITS (theme + world)
- SUPPORT METAPHORS (optional)
- IMAGERY BANK (5–12 motifs)
- MIXED METAPHOR FORBIDDENS
- DEPLOYMENT POINTS (where used: intro/turning/climax)

---

## PROCEDURE

1) Pick core metaphor that matches theme
2) Build imagery bank
3) Align with symbol map (no conflicts)
4) Set forbiddens (avoid mixed metaphor chaos)
5) Deploy only at high-significance beats

---

## VALIDATION RULES

- ME1: Metaphor must clarify theme, not obscure plot.
- ME2: Imagery must fit world law (no modern metaphors in ancient realm unless intended).
- ME3: Avoid mixed metaphors in same scene.
- ME4: Metaphor must be consistent across arc.
