# ENGINE_LIBRARY
## Библиотека знаний движков Universe Engine

---

## Назначение / Purpose

### RU
ENGINE_LIBRARY — это единая каноническая база знаний для ВСЕХ движков.
Она содержит методы, паттерны, чеклисты, термины и тесты, которые:
- не дублируются
- переиспользуются
- масштабируются
- читаются ИИ как нормативная база

### EN
ENGINE_LIBRARY is the canonical knowledge base for all engines.
It stores reusable logic units used by engines without duplication.

---

## Принцип работы

- Движки НЕ хранят знания
- Движки ССЫЛАЮТСЯ на знания
- Знания живут ТОЛЬКО здесь

---

## Типы документов

1. METHODS — как работает логика
2. CHECKLISTS — как проверять
3. TEMPLATES — как формировать
4. PATTERNS — как повторяется
5. EXAMPLES — как выглядит
6. TEST_CASES — как ломается
7. TERMS — как называются вещи
