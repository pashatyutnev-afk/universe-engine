# DOMAIN WORLD ENGINES

CLASS_ID: DOMAIN_WORLD
LEVEL: L2
ROLE: World Logic

## PURPOSE
Формируют законы и структуру мира.

## SCOPE (ALLOWED)
- Экономика
- Геополитика
- Экология
- Хронология

## FORBIDDEN
- Нарративные решения
- Генерация событий

## ALLOWED CALLERS
- Orchestrators

## TYPICAL ENGINES
- World Structure Engine
- Timeline & Epoch Engine
- Economy & Resource Engine

## COMMON MISTAKES
- Использование для сюжета

## CREATION RULE
Описывается закон мира, а не события.
