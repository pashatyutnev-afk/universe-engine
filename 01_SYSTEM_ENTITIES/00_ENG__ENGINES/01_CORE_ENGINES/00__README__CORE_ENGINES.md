# CORE ENGINES

CLASS_ID: CORE_ENGINES
LEVEL: L1
ROLE: System Core

## PURPOSE
Core Engines формируют неприкосновенное ядро системы Universe Engine.
Они определяют законы, память, канон и целостность системы.

## SCOPE (ALLOWED)
- Фиксация фактов и событий
- Управление каноном и правилами
- Обеспечение целостности и памяти
- Блокировка нарушений системы

## FORBIDDEN
- Генерация контента
- Интерпретация смысла
- Работа с нарративом, персонажами, миром
- Форматирование и продакшн

## ALLOWED CALLERS
- System
- Governance Engines

## TYPICAL ENGINES
- Audit Log Engine
- Canon Authority Engine
- Rule Hierarchy Engine
- Versioning & Memory Engine

## COMMON MISTAKES
- Использование Core Engine для логики истории
- Попытка “улучшить” или оптимизировать ядро

## CREATION RULE
Новый Core Engine допускается ТОЛЬКО при необходимости
системного закона или памяти. Любая ошибка = критическое нарушение.
