# DOMAIN CHARACTER ENGINES

CLASS_ID: DOMAIN_CHARACTER
LEVEL: L2
ROLE: Character Logic

## PURPOSE
Определяют психологию, мотивацию и эволюцию персонажей.

## SCOPE (ALLOWED)
- Психология
- Мотивация
- Ценности
- Отношения

## FORBIDDEN
- Написание диалогов как текста
- Нарративное управление сценами

## ALLOWED CALLERS
- Orchestrators
- Narrative Engines

## TYPICAL ENGINES
- Character Core Engine
- Motivation & Desire Engine
- Character Psychology Engine

## COMMON MISTAKES
- Смешивание с Narrative Engines
- Генерация реплик

## CREATION RULE
Каждый движок = один аспект персонажа.
