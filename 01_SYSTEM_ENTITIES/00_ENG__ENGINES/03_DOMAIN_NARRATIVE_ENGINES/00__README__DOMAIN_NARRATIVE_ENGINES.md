# DOMAIN NARRATIVE ENGINES

CLASS_ID: DOMAIN_NARRATIVE
LEVEL: L2
ROLE: Narrative Logic

## PURPOSE
Определяют структуру, форму и законы повествования.

## SCOPE (ALLOWED)
- Драматическая структура
- Логика истории
- Ритм и напряжение
- Темы и смыслы

## FORBIDDEN
- Написание сцен и диалогов
- Генерация текста
- Управление форматами

## ALLOWED CALLERS
- Orchestrators

## TYPICAL ENGINES
- Dramatic Arc Engine
- Narrative Logic Engine
- Pacing & Rhythm Engine
- Theme & Meaning Engine

## COMMON MISTAKES
- Попытка писать историю
- Управление персонажами напрямую

## CREATION RULE
Новый движок описывает ЗАКОН истории, а не её содержание.
