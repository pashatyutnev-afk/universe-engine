Domain Narrative Engines define **how stories are constructed** inside the Universe Engine system.

They provide:
- narrative logic primitives
- story structure primitives
- scene and pacing composition rules
- continuity and theme enforcement mechanisms

These engines do not create system canon.
They create **domain outputs** that must still pass validation/governance if they become binding.
