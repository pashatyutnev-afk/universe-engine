## 0) PURPOSE
This folder defines the final deliverable shapes (“Output Packs”) for VOCAL_DIRECTOR.

Output packs:
- are composed of stable artifact blocks
- must pass gates (G1..G9) depending on tier
- must avoid scope violations (no lyric rewrites, no mix/master chains)

---

## 1) PACK TIERS (MIN / STD / FULL)
### 1.1 MIN (FAST DRAFT)
Goal:
- minimum usable direction for a quick run.

Expected gates:
- must PASS: G1, G2, G9
- should PASS: G3
- optional: G4..G8

### 1.2 STD (DEFAULT)
Goal:
- stable working pack for iterative generation and testing.

Expected gates:
- must PASS: G1..G3, G4, G6, G9
- should PASS: G5, G7, G8

### 1.3 FULL (READY)
Goal:
- complete “READY” pack used as canonical production reference.

Expected gates:
- must PASS: G1..G9
