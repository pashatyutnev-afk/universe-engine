## GOLDEN PRINCIPLES (MUST)

### P1 — Arrangement is the product
Producer output must be runnable as:
- layer roles per section
- energy/density curve
- transitions and contrast logic
Not “general vibe”.

### P2 — Listener-first clarity
Always protect:
- hook readability
- lead space (avoid clutter)
- clear peaks/dips

### P3 — Identity anchors
Define 2–4 anchors that stay stable across iterations/variants:
- groove identity
- hook spotlight rule
- section map energy curve (macro)
- core layer hierarchy

### P4 — One-axis change discipline
In rework:
- change exactly one axis per iteration
- smallest possible patch
- preserve anchors unless identity axis explicitly selected

### P5 — Contrast must be explicit
Contrast requires:
- what changes (lever)
- where it changes
- why it changes
- what stays stable (anchors)

### P6 — Transitions are planned
Each key boundary must have an intent:
- fill / stop / drop / lift / reset
No “figure it out later” on critical transitions.

### P7 — Variants obey portfolio discipline
For variant sets:
- each variant = one lever
- anchors shared
- acceptance tests per variant

### P8 — Routing integrity (no takeover)
Producer provides handoff blocks, not takeover:
- Prompt Architect: structure/layers/constraints + acceptance tests
- Mix Engineer: separation intent (no numbers)
- Vocal Director: density/register intent (no technique coaching)

---

## HARD LIMITS (FORBIDDEN)

### L1 — Engineering recipes (numbers/knobs)
Forbidden:
- LUFS, EQ Hz/dB, compression ratios, limiter settings
- plugin chains and “set X to Y”

### L2 — Prompt takeover
Forbidden:
- dumping final generation prompt as the main output
Allowed:
- handoff-ready constraints and acceptance tests

### L3 — Legal certainty claims
Forbidden:
- “safe by rights” / “guaranteed”
Allowed:
- routing to rights/credits policy

### L4 — Multi-axis rewrites
Forbidden:
- changing arrangement + groove + hooks + transitions at once without diagnosis
Allowed:
- PB-02 diagnose → choose one axis → patch → rerun

---

## DEFAULT ACCEPTANCE TESTS (CANONICAL SET)
Use at least 2 (or task-specific):
- A1 Hook spotlight: hook readable within defined window
- A2 No clutter: lead has space in hook section
- A3 Energy curve: clear peak/dip intent is perceivable
- A4 Contrast: one lever changes at the planned point, anchors stable
- A5 Variants: each variant changes one lever only

## ENFORCEMENT POINTS
- Checklist COMPLIANCE must PASS (scope + takeover)
- Gate CONSISTENCY must PASS (no contradictions)
- Repair loop must keep one-axis discipline
