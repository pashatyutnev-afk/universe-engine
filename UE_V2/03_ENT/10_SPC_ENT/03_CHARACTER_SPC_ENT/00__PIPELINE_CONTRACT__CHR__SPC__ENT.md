## [M] PURPOSE
Step-run contract for CHARACTER_SPC_ENT.
Resolves character specialists by KEY via INDEX_MANIFEST and selects minimal set per task.

## [M] HARD_RULES
- No RAW inside this contract.
- Resolve targets via INDEX_MANIFEST only.
- Minimal opens: only 1–3 specialists unless task explicitly requests full board.

## [M] REQUIRED_KEYS (must exist in INDEX_MANIFEST)
- INDEX_MANIFEST
- PIPELINE_CONTRACT
- SPC.CHR.CHARACTER_ARCHITECT
- SPC.CHR.PERSONALITY_PSYCHOLOGIST
- SPC.CHR.TRAUMA_MOTIVATION_DESIGNER
- SPC.CHR.RELATIONSHIP_DYNAMICS_DESIGNER
- SPC.CHR.DIALOGUE_BEHAVIOR_ANALYST
- SPC.CHR.CHARACTER_EVOLUTION_SUPERVISOR

## [M] ROLE_SELECTOR (deterministic)
- ARCH: [SPC.CHR.CHARACTER_ARCHITECT]
- PSY: [SPC.CHR.PERSONALITY_PSYCHOLOGIST]
- MOTIVE: [SPC.CHR.TRAUMA_MOTIVATION_DESIGNER]
- REL: [SPC.CHR.RELATIONSHIP_DYNAMICS_DESIGNER]
- DLG: [SPC.CHR.DIALOGUE_BEHAVIOR_ANALYST]
- ARC: [SPC.CHR.CHARACTER_EVOLUTION_SUPERVISOR]
- CORE: [SPC.CHR.CHARACTER_ARCHITECT, SPC.CHR.PERSONALITY_PSYCHOLOGIST]
- FULL: [SPC.CHR.CHARACTER_ARCHITECT, SPC.CHR.PERSONALITY_PSYCHOLOGIST, SPC.CHR.TRAUMA_MOTIVATION_DESIGNER, SPC.CHR.RELATIONSHIP_DYNAMICS_DESIGNER, SPC.CHR.DIALOGUE_BEHAVIOR_ANALYST, SPC.CHR.CHARACTER_EVOLUTION_SUPERVISOR]

## [M] STEP-RUN (high level)

S0) SANITY
- ensure TASK_TEXT exists
- resolve INDEX_MANIFEST via KEY: INDEX_MANIFEST
- validate REQUIRED_KEYS exist

S1) SELECT
- if CHR_HINT missing -> default WORK_SET = CORE
- else map via ROLE_SELECTOR
- cap WORK_SET to 1–3 keys unless MODE=MASTERPIECE or CHR_HINT=FULL

S2) OPEN
- open selected specialists by KEY (resolve via INDEX_MANIFEST)

S3) OUTPUT
- emit CHR_SPC_TOKEN:
  - selected_keys
  - next: "го"
