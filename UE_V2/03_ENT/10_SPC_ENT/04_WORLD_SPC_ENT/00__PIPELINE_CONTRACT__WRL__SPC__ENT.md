## [M] PURPOSE
Step-run contract for WORLD_SPC_ENT.
Resolves world specialists by KEY via INDEX_MANIFEST and selects minimal set per task.

## [M] HARD_RULES
- No RAW inside this contract.
- Resolve targets via INDEX_MANIFEST only.
- Minimal opens: only 1–3 specialists unless task explicitly requests full board.

- INDEX_MANIFEST
- PIPELINE_CONTRACT
- SPC.WRL.WORLD_BUILDER

- SPC.WRL.RELIGION_MYTHOLOGY_DESIGNER

## [M] STEP-RUN (high level)

S0) SANITY
- ensure TASK_TEXT exists
- resolve INDEX_MANIFEST via KEY: INDEX_MANIFEST
- validate REQUIRED_KEYS exist

S1) SELECT
- if WRL_HINT missing -> default WORK_SET = CORE
- else map via ROLE_SELECTOR
- cap WORK_SET to 1–3 keys unless MODE=MASTERPIECE or WRL_HINT=FULL

S2) OPEN
- open selected specialists by KEY (resolve via INDEX_MANIFEST)

S3) OUTPUT
- emit WRL_SPC_TOKEN:
  - selected_keys
  - next: "го"
