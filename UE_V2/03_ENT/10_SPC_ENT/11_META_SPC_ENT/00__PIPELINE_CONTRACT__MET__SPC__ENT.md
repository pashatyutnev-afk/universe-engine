## [M] PURPOSE
Step-run contract for META_SPC_ENT.
Resolves meta specialists by KEY via INDEX_MANIFEST and selects minimal set per task.

## [M] HARD_RULES
- No RAW inside this contract.
- Resolve targets via INDEX_MANIFEST only.
- Minimal opens: 1–3 specialists unless task explicitly requests full board.

## [M] REQUIRED_KEYS (must exist in INDEX_MANIFEST)
- INDEX_MANIFEST
- PIPELINE_CONTRACT
- SPC.MET.UNIVERSE_CURATOR
- SPC.MET.SYSTEM_ARCHITECT
- SPC.MET.RULE_SYSTEM_DESIGNER
- SPC.MET.DEPENDENCY_SUPERVISOR
- SPC.MET.KNOWLEDGE_INTEGRATOR
- SPC.MET.META_ANALYST
- SPC.MET.LONG_TERM_STRATEGY_PLANNER
- SPC.MET.EVOLUTION_SCALING_PLANNER

## [M] ROLE_SELECTOR (deterministic)
- CANON: [SPC.MET.UNIVERSE_CURATOR]
- ARCH: [SPC.MET.SYSTEM_ARCHITECT]
- RULE: [SPC.MET.RULE_SYSTEM_DESIGNER]
- DEP: [SPC.MET.DEPENDENCY_SUPERVISOR]
- KB: [SPC.MET.KNOWLEDGE_INTEGRATOR]
- ANALYTICS: [SPC.MET.META_ANALYST]
- STRAT: [SPC.MET.LONG_TERM_STRATEGY_PLANNER]
- SCALE: [SPC.MET.EVOLUTION_SCALING_PLANNER]
- CORE: [SPC.MET.UNIVERSE_CURATOR, SPC.MET.SYSTEM_ARCHITECT]
- FULL: [SPC.MET.UNIVERSE_CURATOR, SPC.MET.SYSTEM_ARCHITECT, SPC.MET.RULE_SYSTEM_DESIGNER, SPC.MET.DEPENDENCY_SUPERVISOR, SPC.MET.KNOWLEDGE_INTEGRATOR, SPC.MET.META_ANALYST, SPC.MET.LONG_TERM_STRATEGY_PLANNER, SPC.MET.EVOLUTION_SCALING_PLANNER]

## [M] STEP-RUN (high level)

S0) SANITY
- ensure TASK_TEXT exists
- resolve INDEX_MANIFEST via KEY: INDEX_MANIFEST
- validate REQUIRED_KEYS exist

S1) SELECT
- if MET_HINT missing -> default WORK_SET = CORE
- else map via ROLE_SELECTOR (cap to 1–3 keys unless MODE=MASTERPIECE or MET_HINT=FULL)

S2) OPEN
- open selected specialists by KEY (resolve via INDEX_MANIFEST)

S3) OUTPUT
- emit MET_SPC_TOKEN:
  - selected_keys
  - next: "го"
