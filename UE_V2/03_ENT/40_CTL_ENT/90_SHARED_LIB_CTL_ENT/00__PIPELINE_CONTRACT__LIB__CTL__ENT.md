## [M] PURPOSE
PIPELINE_CONTRACT — роутер общей библиотеки CTL.
Возвращает нужные KEY для подключаемых политик: negative-spec, safe-copy, output naming.
RAW не хранит: всё через KEY и INDEX_MANIFEST.

## [M] HARD_RULES
- RAW запрещён внутри CONTRACT.
- Все ссылки: только KEYS.
- MODE REPO: никаких правок репозитория.
- Выход: CTL_LIB_PLAN (KEYS) + APPLY_NOTES.

## [M] REQUIRED_KEYS
- INDEX_MANIFEST
- PIPELINE_CONTRACT
- CTL.LIB.NEGATIVE_SPEC_LIBRARY
- CTL.LIB.SAFE_COPY_POLICY
- CTL.LIB.OUTPUT_NAMING_POLICY

## [M] DEFAULT_PLAN (keys)
- ALWAYS:
  - CTL.LIB.SAFE_COPY_POLICY
  - CTL.LIB.OUTPUT_NAMING_POLICY
- CONDITIONAL:
  - If prompt/lyrics/video/web outputs -> CTL.LIB.NEGATIVE_SPEC_LIBRARY
