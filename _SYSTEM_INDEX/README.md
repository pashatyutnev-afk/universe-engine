# Universe Engine
## Движок Вселенных

A universal system engine for creating, managing and evolving universes, projects and meaning structures.

Универсальный системный движок для создания, управления и развития
вселенных, проектов и смысловых структур.

---

### Core Principles / Основные Принципы
- System over improvisation / Система важнее импровизации
- Structure before content / Структура раньше контента
- Canon over chaos / Канон важнее хаоса
- Scalability without rewriting / Масштабирование без переписывания
- One engine — infinite projects / Один движок — бесконечные проекты

---

### Status / Статус
Initialization phase.
Structure first. Content later.

Фаза инициализации.
