# 09_ENTITIES
## Canon Entity Registry / Реестр канонических сущностей

---

## 1. Назначение

Папка **09_ENTITIES** является централизованным реестром
всех канонически утверждённых сущностей,
появившихся в рамках конкретных проектов.

Entities **НЕ являются генеративным слоем**.
Они не создаются сами по себе и не используются для идей.

Entities — это **зафиксированный результат проектной работы**.

---

## 2. Ключевой принцип

> **ENTITIES = ТО, ЧТО УЖЕ СУЩЕСТВУЕТ В ПРОЕКТЕ**

Сущность может быть добавлена в 09_ENTITIES
ТОЛЬКО после того, как она:

- появилась в проекте (текст, сцена, серия, глава)
- была использована в нарративе
- признана каноном проекта

---

## 3. Что НЕ является Entities

09_ENTITIES НЕ используется для:
- генерации идей
- черновиков
- вариантов «на подумать»
- творческих экспериментов

Для этого существуют:
- ENGINES
- SPECIALISTS
- PROJECTS

---

## 4. Роль в системе Universe Engine

Entities являются:

- источником стабильных ссылок
- основой канона
- защитой от расхождений
- якорем идентичности мира

Через Entities система:
- помнит персонажей
- не меняет объекты
- не дублирует события
- сохраняет целостность мира

---

## 5. Поток данных (канонический)


- ENGINES задают правила
- SPECIALISTS принимают решения
- PROJECTS создают контент
- ENTITIES фиксируют результат

---

## 6. Категории сущностей

Entities группируются по типу:

- Characters — персонажи
- Locations — места
- Objects — предметы
- Creatures — существа
- Technologies — технологии
- Artifacts — уникальные объекты
- Organizations — фракции, структуры
- Concepts — абстрактные сущности
- Events — события
- Systems — внутримировые системы

---

## 7. Работа с Entities

Entities:
- добавляются вручную или по команде
- имеют паспорт сущности
- не изменяются без фиксации версии
- используются как каноническая ссылка

---

## 8. Важное правило

> **Если сущность не зафиксирована в 09_ENTITIES —
она не считается каноном.**

---

## 9. Статус

- Тип: Canon Registry
- Уровень: Project Output
- Изменяемость: Контролируемая
- Назначение: Долгосрочная стабильность мира

---

## 10. Примечание

Entities — это память мира.
Без памяти мир не развивается —
он распадается.
